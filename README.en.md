# Pomodoro

#### Description

本软件为本人开发的番茄钟时间管理软件，在番茄计时器的基础上增加了本人时间管理中需要的多种功能。

#### Corporation

软件开发遇到如下问题，现寻求合作：

1. 本人暂时不了解网络相关的开发，难以进行跨设备同步操作；

2. 受设备限制，尽管Qt具有跨平台的能力，现阶段仅能编译出Windows下的桌面软件，移动端难以编译与测试。

希望与您的合作可以让我们共同解决这些问题。有合作开发意向的可以发送邮件至cyk_phy@mail.ustc.edu.cn联系合作。
